# BackEnd - Server

Este es el código para establecer el backend conectandose a mongoDB usando mongoose.

Para ejecutarlo es necesario reconstruir los modulos de node usando el comando
```
npm install
```

## Demo de google-signin-demo

Existe un pequeño ejercicio para probar la autenticacion de Google en un frontend basico pero funcional, el cual al logearse con google en consola muestra el token del usuario.
